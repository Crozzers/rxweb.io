Before talking about the coming new feature, we would like to share about our long-term vision for rxweb in open source.

Our vision for rxweb framework is...

Core Values:

  • Focus on those features which are not available in Angular framework or taking more efforts to achieve with the standard approach.
  • Issues and pull requests get handled within reasonable time as per our issue workflow.
  • Always willing to help or guidance.
  • Always welcome new comers and respect their valuable effort as experts.
  • Seeking to improve ourselves and our community.
  • Recognize and validate contributions.

Stable features:

  • Easy to understand and implement advance features for our community devs.

Excellent Documentation:

  • Focus on every API should be documented with working code example for better user experience.
  • Up to date API and reference docs.

Reliability:

To prevent bugs and ensure stable release, rxweb framework heavily rely on tests and as a practice every new functionality or bug resolution test must be written with their respective scenarios. This practice helps to ensure while developing new features in existing features without any bug.
