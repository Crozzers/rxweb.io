1.9.0

add value property in greaterThan and greaterThanEqualTo validator config

New value property in greaterThan and greaterThanEqualTo through configuring it the validation can be done based upon a static value.

add value property in lessThan and lessThanEqualTo validator config

New value property in lessThan and lessThanEqualTo through configuring it the validation can be done based upon a static value.

add a new method of commit in RxFormGroup

commit method is used to update the value of the baseObject in RxFormGroup, It will take the whole snapshot of the FormControls.

add new property of isModified in RxFormGroup

isModified is used to check whether there are any changes in the formControls.

add new property of modifiedValue in RxFormGroup

modifiedValue is used to patch value by sending only the data which is changed instead of sending the whole form data.

dynamically set default value in propconfig

To set the default value dynamically from the component at any point in time.

Exclude properties conditionally

To exclude properties dynamically based on specific condition.

Extend feature of resetForm method

New features of reseting formControls,formGroups and formArrays including nested level.

minTime and maxTime Validations

New minTime and maxTime validations with operator and value properties.

ignore Undefined Props

This is used to ignore the undefined properties dynamically.
