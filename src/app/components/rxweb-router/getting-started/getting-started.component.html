It is used to manage navigation in single page applications along with determining user's access, resolving authentication, using middleware.

Install Package

you need to install @rxweb/angular-router

Creating RxWebModule

RxWebModule is made for using multiple configurations and using the services of the router package which resolves all the necessary information of the user from the server which is useful after navigating done which the help of router. you first need to create RxWebModule and create @routerModule

The router configuration properties throughout the application which can be used for the functionalities as listed below.

After creation of RxWebModule, it needs to be imported in app module

  1. Authentication for resolving the authentication details from the server.
  2. For resolving the access from the server for a particular user Authorization
  3. For performing any action at the time of request calling Middleware
  4. Url Encryption for encrypting the url.

The above code is of rxweb.module.ts from which RxWebModule is imported in the app module.

BaseCanActivate

Routing enables you to navigate the path from one route to another. For more information about routing, visit Angular Routing While declaring routes, we provide the path and the component name to the respective component name To route the configured services at the time of navigation use the BaseCanActivate. It will allow to use the @routerModule services while writting any custom code or adding any complex logic.
