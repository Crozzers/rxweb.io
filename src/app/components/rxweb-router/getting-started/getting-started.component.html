It is used to manage routing in single page applications along with providing access, managing middleware.

Install Package

you need to install @rxweb/angular-router

Creating RxWebModule

For using multiple configurations and using the services of the router package, you first need to create `RxWebModule` and create @routerModule

It is used to configure router configuration properties throughout the application which can be used for the following functionalities.

  1. Authentication for resolving the authentication details from the server.
  2. For resolving the access from the server for a particular user Authorization
  3. For performing any action at the time of request calling Middleware
  4. Url Encryption for encrypting the url.

The created RxWebModule is as below, register it into the app module as an import.

BaseCanActivate

To route the configured services at the time of navigation use the BaseCanActivate.
