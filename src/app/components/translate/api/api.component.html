Parameters, options and usage of the decorators, services and directives of @rxweb translate

@translate

translate decorator is used to initialize the multilingual content onject through which the translation content is appended into the html.

ParameterDescription
translationNametranslation name is based upon the multilingual content to be resolved from the i18n folder based upon the language code. For example : if you want to retrieve information from user.en.json the translation name should be user.
languagelanguageCode in case of fixed language content
filePathThe custom path of the json file from which the multilingual content is resolved.

rxTranslate

rxTranslate directive binds the translation content in the shared components

ParameterDescription
[rxTranslate]For Binding the translation content rxTranslate paramater is set true.

@asyncTranslate

@asyncTranslation is used to perform async translation based upon the retrieval of server data.

ParameterDescription
serviceModelName of the custom service.
serviceMethodName of the method of the service which is used for fetching the data.

RxTranslateModule

RxTranslateModule is the global configuration used for cache related configurations and path of the multilingual data.

ParameterDescription
languageCodeLanguage code.
filePathmultilingual json file path.
cacheActiveLanguageObjectStore active language cache set to true or false.
cacheLanguageWiseObjectStore language wise cache set to true or false.
globalFilePathglobal multilingual file path.
preloadingStrategyPre laod module for translation set to true or false

RxTranslation

RxTranslation is a service used for translating content and for changing language while programmable language change.

ParameterDescription
changeTo change the language code.
translateTo translate the text.
